source code 

DRY helper function for SQLTABLEs in CRUD

Parameters:
  • fields - list of fields to display
  • left - left outer joins
  • start - index of the first record to display
  • limit - maximum number of records to display
  • orderby - orderby for the query
  • linkto - hook to link record IDs
  • download_url - the default download URL of the application
  • as_page - return the list as JSON page
  • as_list - return the list as Python list
  • format - the representation format
