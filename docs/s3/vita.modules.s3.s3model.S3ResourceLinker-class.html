Get a query for the origin table to retrieve records that are linked to a set of target table records.

Parameters:

  • to_table - the target table
  • to_id - target record ID or list of target record IDs
  • link_class - link class name
  • union - retrieve a union (True) or an intersection (False, default) of all sets of links (in case of multiple target records)
