source code 

Push (=POST) the current resource to a target URL

Parameters:
  • url - the URL to push to
  • template - path to the XSLT stylesheet to be used by the exporter
  • as_json - convert the output into JSON before push
  • xsltmode - "mode" parameter for the XSLT stylesheet
  • start - index of the first record to export (slicing)
  • limit - maximum number of records to export (slicing)
  • marker - default map marker URL
  • msince - export only records which have been modified after this datetime
  • show_urls - show URLs in the <resource> elements
  • dereference - include referenced resources in the export
  • content_type - content type specification for the export
  • username - username to authenticate at the peer site
  • password - password to authenticate at the peer site
  • proxy - URL of the proxy server to use

limitby(self, start=None, limit=None)

source code 

Convert start+limit parameters into a limitby tuple

  • limit without start => start = 0
  • start without limit => limit = ROWSPERPAGE
  • limit 0 (or less) => limit = 1
  • start less than 0 => start = 0
Parameters:
  • start - index of the first record to select
  • limit - maximum number of records to select
