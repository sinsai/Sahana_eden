rlevenshtein(self, str1, str2)

source code 

Returns a relative value for the Levenshtein distance of two strings

Levenshtein distance: Number of character insertions/deletions/replacements necessary to morph one string into the other. Here calculated in relation to the maximum string length, hence 0.0 for identical and 1.0 for completely different strings.

Note: Unfortunately, this doesn't work in SQL queries :(
