source code 

Push (=POST) this resource as XML to a target URL

Parameters:
  • url - the URL to push to
  • template - path to the XSLT stylesheet to use
  • xsltmode - XSLT stylesheet "mode" parameter
  • username - username for HTTP basic auth (optional)
  • password - password for HTTP basic auth (optional)
  • proxy - proxy server to use (optional)
  • start - start record (for pagination)
  • limit - maximum number of records to send (for pagination)
  • marker - path to the default marker for GIS features
  • msince - export only records modified after that datetime (ISO-format)
  • show_urls - show URLs in resource elements
  • dereference - export referenced objects in the tree
Returns:
the response from the peer as string
